# 0.5.2 — 2026-05-23

## Changed

- **`@cyanheads/mcp-ts-core` `^0.9.1 → ^0.9.6`** — picks up all upstream fixes across the series: OTel attribute type corrections, context-state key-lookup improvements, `spillover()` reliability fixes, and linter rule additions (`server-json-*`, `schema-format-portability`).
- **`RequestContextLike` canvas cast removed** — `CanvasBridge` now receives a concrete `Context` directly; the intermediate cast was unnecessary after the framework stabilized `RequestContextLike`'s exported shape.
- **`zod` added as explicit dependency** — was previously only a transitive dep from the framework; now declared in `dependencies` for `z` re-exports used in tool schemas.
- **`manifest.json` + `.mcpbignore` scaffolded** — enables `bun run bundle` to produce a `.mcpb` extension bundle for one-click Claude Desktop install via `@anthropic-ai/mcpb`.
- **Install badges added to README** — Visual Studio Code and Claude Desktop one-click install buttons.

# 0.7.2 — 2026-05-31

## Added

- **Local mirror (opt-in)** — set `EDGAR_MIRROR_ENABLED=true` to route `resolveCik`, `tryGetCompanyConcept`, and `tryGetFrames` to a local SQLite mirror of `company_tickers.json` + the `companyfacts.zip` bulk archive. Live-API fallback on miss controlled by `EDGAR_MIRROR_FALLBACK_LIVE` (default `true`). ([#21](https://github.com/cyanheads/secedgar-mcp-server/issues/21))
- **`EDGAR_MIRROR_PATH`** — directory for the mirror SQLite databases (default `./data/edgar-mirror`).
- **`EDGAR_MIRROR_REFRESH_CRON`** — in-process nightly refresh schedule for HTTP transport (e.g. `0 9 * * *`); omit to refresh out-of-band.
- **`EDGAR_MIRROR_FALLBACK_LIVE`** — set `false` for strict mirror-only reads.
- **`bun run mirror:init`** — one-time bootstrap: downloads `company_tickers.json` + streams `companyfacts.zip` into SQLite. Resumable.
- **`bun run mirror:refresh`** — incremental refresh from SEC bulk files.
- **`bun run mirror:verify`** — prints sync status and runs sample reads.
- **`better-sqlite3 ^12.10.0`** and **`fflate ^0.8.2`** — new production deps for the mirror store and zip streaming.
